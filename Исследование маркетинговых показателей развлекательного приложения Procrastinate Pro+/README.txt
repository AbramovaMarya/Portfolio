Исследование маркетинговых показателей развлекательного приложения Procrastinate Pro+

Описание
Для команды развлекательного приложения Procrastinate Pro+ необходимо провести исследование маркетинговых показателей, определить возможные причины убытков компании несмотря на проведение рекламных акций.

В рамках исследования предстоит изучить:

- откуда приходят пользователи и какими устройствами они пользуются,
- сколько стоит привлечение пользователей из различных рекламных каналов;
- сколько денег приносит каждый клиент,
- в каких случаях расходы на привлечение клиента окупаются,
- какие факторы мешают привлечению клиентов.

Инструменты
pandas
numpy
matplotlib
seaborn
обработка данных, статистический тест, LTV, CAC, когортный анализ

Выводы
Для команды развлекательного приложения Procrastinate Pro+ было проведено исследование маркетинговых показателей с целью определения возможных причин убытков компании несмотря на проведение разнообразных рекламных акций. Компания-заказчик представлена на рынке США и некоторых европейских стран (Германия, Франция, Великобритания).
В ходе анализа была проведена предобработка данных, исследовано влияние таких параметров как регион, ОС, поведение органических и акционных пользователей, временные интервалы. Рассчитаны маркетинговые показатели для каждого параметра, предложены рекомендации.
