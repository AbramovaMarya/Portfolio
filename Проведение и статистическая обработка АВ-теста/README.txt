Проведение и статистическая обработка АВ-теста

Описание 
Отдел маркетинга крупного интернет-магазина подготовил список гипотез для увеличения выручки и провёл А/В-тест одной из них. приоритезирвать гипотезы, применив фреймворк ICE/RICE, и объяснить результат; проанализировать результат, принять аргументированное решение об остановке или продолжению теста.

Задачи:
- приоритезирвать гипотезы, применив фреймворк ICE/RICE, и объяснить результат;
- проанализировать результат запущенного А/В-теста, принять аргументированное решение об остановке или продолжении теста.

Инструменты
pandas
numpy
matplotlib
seaborn
scipy.stats, binom

Выводы
Проведена приоритизация гипотез по фреймворкам ICE и RICE. Был проведен анализ результатов A/B-теста, построены графики кумулятивной выручки, среднего чека, конверсии по группам, посчитана статистическую значимость различий конверсий и средних чеков по сырым и очищенным данным. На основании анализа было принято решение о нецелесообразности дальнейшего проведения теста, даны дополнительные рекомендации по работе с постоянными клиентами.
